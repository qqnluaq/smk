    {{ tool.icon }} Search Infestations Tool

    The Search Infestations Tool is a custom tool for SMK Forest Health that allows user to search with a common name, and a severity. Once the search completes the features will be labelled on the map with a number displaying how many features are at that location.

    The maximum number of features that can be found at once is ten-thousand so if you’re frequently hitting that limit you may want to pare down your search parameters. Ideally you’ll be using a name and a severity.

    Once you’ve found a feature and obtained the information you want from it, hit the back arrow in the upper left to return to the search screen and press reset to reset the search terms.
